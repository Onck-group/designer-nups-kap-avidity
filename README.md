# designer-nups-kap-avidity
Simulation files for the paper "Spatial control of karyopherin binding avidity within NPC mimics revealed by designer FG-Nucleoporins"

Files are present for:
- Design approach
- Sequence analysis
- Single-protein Rs + KD simulations
- Nanopores with and without cargo
- Bindign simulations

Virtually all analyses or setups make use of the following softwares:
- Shell scripts (compatible with unix-based systems)
- GROMACS 2018.1 or 2018.4 on a linux-based high-performance computing cluster
- Python version > 2.8 using numpy and MDAnalysis 2.0
- MATLAB R2018b or later

Specific usage instructions for the various scripts, analyses and simulation setups can be found in README files or script preambles throughout the individual sub-directories.
